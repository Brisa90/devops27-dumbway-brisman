## Perbedaan Shell dan Bash(Bonus again shell)
Shell adalah istilah umum untuk antarmuka penerjemah perintah (command-line interpreter) di Linux, sedangkan Bash (Bourne Again SHell) adalah jenis shell spesifik yang paling populer dan default di hampir semua distro Linux. Shell adalah konsepnya, dan Bash adalah salah satu implementasinya yang canggih
## Fitur dan kemampuannya
- Shell (Umum/Dasar): Beberapa shell lama (seperti sh asli) memiliki fitur terbatas dan hanya menyediakan fungsionalitas dasar.
- Bash (Lanjutan): Menyediakan fitur interaktif yang kuat, seperti:
  - Autocompletion: Menekan Tab untuk melengkapi perintah atau nama file.
  - Command History: Kemampuan memanggil perintah sebelumnya.
  - Scripting Canggih: Mendukung sintaksis pemrograman yang lebih kompleks dibanding 

## Command Linux dan fungsinya
- cp = digunakan untuk copy paste 
- ls = menlihat direktori
- cd = pindah direktori
- echo = menampilkan pesan sebagai output
- pwd = menampilkan path direktori saat ini
- mkdir = membuat direktori
- rm = menghapus file
- mv = memindahkan atau mengganti nama file dan direktori
- touch = membuat file kosong baru
- file = memeriksa jenis file
- zip and unzip =  membuat mengekstrak arsip ZIP
- tar mengarsipkan file tanpa kompresi dalam format TAR
- nano, vi, and jed = mengedit file dengan editr teks
- cat = mencantumkan, menggabungkan dan menulis isi fikle dengan output standar
- grep = mencari string di dalam file
- sed = menemukan, mengganti, atau menghapus pola dalam file
- head = menampilkan sepuklluh barius pertama ffile
- tail = menampilkan sepuluh baris terakhir file
- awk = memanipulasi dan menemukan pola dalam file
- cut = membagi dan menampilkan baris dari sebuah file
- diff = membandingkan isi dari dua file dan perbedaannya
- tee = menampilkan ooutput perintah di terminal dan file
- locate = menemukan file dalam database sistem
- sudo = menjalankan perintah super user
- su = menjalankan rpogram di shell saat ini sebagai user lain
- useradd dan userdel = menambahkan dan menghapus user
- systemctl = mengelola layanan sistem
- shutdown = mematikan atau memulai ulang sistem
- pinggg = memeriksa konektivitas jaringan
- nestat menampilkan informasi antaramuuka jaringan sistem
- apt-get -  mengelola library paket distro berbais debuan

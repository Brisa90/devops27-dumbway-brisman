# Task 1: Pengenalan Devops & Install Sistem Operasi

## Pengertian & Tugas Devops
- Bertugas untuk memproses dan menghubungkan antara pengembang(Development) dan operation hingga rilis ke publik

## DevOps Proses
- Plan
- Code
- Build
- Test

- Release
- Deploy
- Operation
- Monitor

## Keuntungan Devops
1. Mempercepat proses realis aplikasi ke publik
2. Mempersingkat waktu perbaikan
3. Mengurangi tingkat ke gagalan rilis terbaru

## Operation System
- menghubungkan antara hardware dan software sehingga dapat digunakan oleh user(pengguna)
## Virtualization
untuk membagi satu resource komputer menjadi beberapa bagian
- Hypervision program yang membagi dan menjalankan Virtual machine
## keunggulan Virtualization
1. Mempermudah Back up dan recovery data
2. Kemuddahan Clone data virtual ke hardware/server lain

## Buat step by step penggunaan text manipulation! (grep, sed, cat, echo)
## 1. penggunaan perintah cat
- untuk membuat, melihat dan menggabungkan file
- membuat file1 & file2 bersetad isi file menggunakan 'cat > file1' & 'cat > file2'
- untuk meilihat menggunakan perintah 'cat file1' 
- untuk menggabungkan kedua file, di file perintah 'cat file1 file2 > fule3'
<img width="973" height="502" alt="2" src="https://github.com/user-attachments/assets/c781e4b3-87a9-4037-894e-f232d6f0016d" />

## 2. penggunaan sed
- untuk mengedit isi file dapat menggunakan perintah "sed -i 's/kata yang ingin diganti/kata baru' nama-file"
<img width="973" height="514" alt="3" src="https://github.com/user-attachments/assets/1f4bd57f-e90a-4964-ba35-70dc7692af50" />

## 3. penggunaan grep
- untuk mencari kata, bisa menggunakan perintah "grep kata-yang-ingin-dicari nama-file"
<img width="976" height="511" alt="4" src="https://github.com/user-attachments/assets/602c4bfa-cee3-46ce-be23-f80c2a088f69" />

- untuk mencari bisa menambah kan "-c" didepan kata yang ingin dicari, seperti ini "grep -c kata-yang-ingin-dicari nama-file"
- jika ingin mencari kata tertentu diseluruh file yang direktori bisa memakai * untuk mengganti hama file " grep kata-yang-ingin-dicari * "
<img width="983" height="504" alt="5" src="https://github.com/user-attachments/assets/4788d3a3-a8f8-42ae-a218-c65a6a33115f" />

## 4. menggunakan Echo
- penggunaan echo digunakan untuk menampilkan pesan sebagai output contoh "echo nama pesan" output "nama pesan'
- penggunaaan echo untuk membuat isi dan file menggunakan perintah "echo -e 'isi file' >> nama-file", maka akan ada file yang berisikan sesuai yang kita isi
